# ⚠️ Warning
Make a backup of your files and your registry before execute this script.
WAFS delete and modify registry keys value and windows services.


# 👷 Installation
```bash
git clone https://github.com/MikeHorn-git/WAFS.git
cd WAFS
.\wafs.ps1
```
